# 论文技术要点备忘
最后同步日期：`2026-02-21`

## 1. 题目与范围

题目建议：
- 面向用户的多模态语音情绪分析与心理状态预警 Web 系统设计与实现。

边界说明：
- 系统输出为情绪风险提示与趋势评分，不做医学诊断结论。

## 2. 可写的技术创新点

- 多模态融合：音频情绪 + 文本语义联合决策。
- 实时可视化：任务阶段流 + 风险曲线 WebSocket 推送。
- 可解释风险：报告页展示语音分、文本分、融合分、PSI 贡献项。
- 模型治理：管理端支持规则、告警、模型切换与审计日志。

## 3. 方法章节可落地结构

1. 系统架构设计（前后端 + SER 服务 + 数据库）。
2. 音频情绪模型（wav2vec2 分类微调）。
3. 文本情绪模型（中英文分路）。
4. Late Fusion 与温度校准。
5. PSI 指标设计与风险分级策略。
6. 实时分析与可视化实现。
